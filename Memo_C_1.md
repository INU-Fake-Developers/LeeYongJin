```
#include <stdio.h>
int main()
{
    return 0;
}`
```
프로그래밍의 기초와 입출력에 대해서 다루겠습니다.
Hello World! 부터 출력해보죠!  
`#include <stdio.h>` 를 통해서 헤더 파일을 호출해줍시다.  
추후에 다루겠지만, .h를 통해 입출력에 기본적으로 필요한  
`scanf`와 `printf` 함수를 사용할 수 있습니다.
